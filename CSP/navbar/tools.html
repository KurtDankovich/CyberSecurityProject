Necessary Tools

Just T-Pot

Home Device

This really can be any home device such as a laptop, desktop, or tablet as long as it has a web browser and internet access as a cloud services needs internet to access.

Internet Connection

An internet connection is needed. It does not have to be the best internet plan however the internet connection needs to work and be secure.

Cloud Platform

A cloud platform is needed in order to get the project up and running. The project is going to be hosted on a cloud service such as AWS, Azure or Google Cloud. Each platform has different free trail offers however I choose Google Cloud as Lewis University gives students $100 in credit on top of the $300 credit for free from Google Cloud.

T-Pot Github Repository

T-Pot's github repository can be found at: https://github.com/telekom-security/tpotce

Linux command line: git clone https://github.com/telekom-security/tpotce.git

Debian 11

T-Pot is ran best on Debian 11 Bullseye. Debian 11 Bullseye was released on August 14th, 2021. It contains over 11,294 new packages including Python 3 (3.9.1), OpenSSH (8.4p1), MariaDB (10.5) and VIM (8.2).

Google Cloud offers Debian 11 Bullseye within their virtual machine operating system configurations, however an .iso file can be downloaded from https://www.debian.org/download. The .iso file is used to configure a virtual machine instance on softwares such as VMWare or VirtualBox.

Deploy Static Web Page

Domain Name

Acquire a domain name from a domain hosting site such as Google Domains, Bluehost, GoDaddy or another creditable site. The domain name will be where the web page can be reached at.

Virtual Studio Code

A html, css, javascript friendly enviroment is needed to create the web page. Virtual Studio Code has built in live server where one can see their page in real time. Virtual Studio Code can be downloaded from https://code.visualstudio.com/download.

Cloud Platform

A cloud platform is needed to host the website. Web pages can be hosted on several cloud service such as AWS, Azure or Google Cloud.

Cloud Bucket

The Google Cloud Platform has cloud buckets which is used to store the contents of a webpage.
